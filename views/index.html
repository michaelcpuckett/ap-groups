chirp.social is a place for groups

What is this?

This is an ActivityPub server that can interact with Mastodon.

Any post addressed to the group by a follower will be boosted automatically by the group account, for all followers to see.

Group Directory

How does this work?

All accounts created here are Group accounts.

Anyone that follows the account will be a group member.

Everything is public and shows up in the timeline.

Group accounts cannot post anything from here currently.

This server is where you login and manage your group.

Currently there is no way to change ownership, moderatation is limited, etc. Please contribute ideas/code on Github
